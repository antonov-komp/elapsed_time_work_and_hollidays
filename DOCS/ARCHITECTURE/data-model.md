## Планируемая модель данных

### Табель присутствия

**Описание:** Основная сущность для хранения данных о присутствии пользователя

#### Поля (предварительно)
- `id` — уникальный идентификатор записи
- `user_id` — ID пользователя Bitrix24
- `date` — дата записи
- `status` — статус присутствия (присутствовал, отсутствовал, больничный, отпуск)
- `hours` — количество отработанных часов
- `comment` — комментарий
- `created_at` — дата создания записи
- `updated_at` — дата обновления записи

#### Связи
- Связь с пользователем Bitrix24 через `user_id`
