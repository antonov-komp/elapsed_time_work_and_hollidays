## Содержание

### [Техническое задание (черновик)](technical-specification.md)

Первоисточник технического задания на разработку приложения "Мой Ежемесячный табель присутствия".

**Статус:** Черновик  
**Версия:** Первоисточник

### [Интерфейс табеля присутствия (Календарный месяц)](interface-timesheet-calendar.md)

Первое детальное техническое задание на интерфейс табеля присутствия с календарным отображением месяца.

**Статус:** Черновик  
**Версия:** 1.1 (Дополнено деталями)

**Описание:**
- Календарный месяц в виде шахматной доски
- Авторизация и отображение ФИО/Должности
- Выбор месяца и года (2025-2035)
- Заполнение рабочих часов с шагом 30 минут
- Особые статусы (Больничный, Командировка, Отпуск)
- Групповое заполнение недели
- Статистика по месяцам
- Хранение данных в JSON (ID_сотрудника/ГОД/МЕСЯЦ/data.json)

## Назначение папки

Папка `TZ/` содержит:
- Первоисточник технического задания

---

## Использование

На основе технического задания создаются задачи в `DOCS/TASKS/`:
- Каждая задача ссылается на соответствующий раздел ТЗ
- Детализация требований происходит в задачах
- ТЗ обновляется при необходимости
